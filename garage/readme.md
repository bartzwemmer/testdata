# Garage

Garage is an S3 compatible storage. This compose starts a 1 node cluster, with a web ui on localhost:3909. The init script creates a bucket and a file in that bucket.

## Ports

| Port | Description |
| ---- | ----------- |
| 3900 | S3          |
| 3901 | API         |
| 3902 | WebUI       |
| 3903 | WebUI       |
| 3909 | WebUI       |
